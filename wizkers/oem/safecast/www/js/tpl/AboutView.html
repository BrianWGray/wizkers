About

Safecast:Drive is the official Android app for the Safecast bGeigie. It will let you connect a bluetooth-equipped bGeigie Nano to your phone/tablet and get a real time visualization of its data. You can also record your drives directly on the app, and upload them to Safecast.

Version 0.4.4

  • Improvements on Drive display and deletion.

Version 0.4.3

  • Attempt at supporting version 2.0.0 of the BLEBee module (but you should really update to 2.0.1).
  • BLEBee firmware upgrader.

Version 0.4.0

  • New: Safecast:Drive can now live stream data to the Safecast api.
  • bGeigie logs are uploaded to the API with a filename that complies with Safecast guidelines.
  • Prevent GPS from shutting down when in background on latest Android versions

Version 0.3.1

  • New guided first use configuration help

Version 0.3.0

  • Support for the Medcom Blue Onyx

Version 0.2.0

  • bGeigie: remove requirement for File access, faster processing and submission of logs.
  • Material Design interface.

Version 0.1.7

  • bGeigie: automatically attempt to reconnect to bGeigie if bluetooth connection drops.
  • Interface layout improvements, performance improvements.
